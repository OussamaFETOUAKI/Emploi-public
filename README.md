Emploi-public is a mobile application designed to help users easily access and consult public job offers.
The application allows users to browse available public employment opportunities in a clear and organized way, making job searching more efficient and accessible.

The project is built with:

A Laravel backend that manages job data and API services.

A React Native mobile application that provides a smooth and user-friendly interface.

Emploi-public aims to simplify the process of finding public sector job opportunities by centralizing information and presenting it in a practical mobile format.
